# 📊 Executive Overview Dashboard (Power BI)

## 🚀 Project Overview

This project is a fully dynamic **Power BI Executive Dashboard** built to analyze business performance across orders, revenue, delivery, and customer behavior.

From raw data to insights — the complete pipeline includes **data cleaning, transformation, modeling, DAX, and visualization**.

---

## 🎯 Objective

* Track overall business performance
* Identify top-performing restaurants
* Analyze delivery efficiency & order status
* Provide decision-ready insights using interactive visuals

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Measures & Calculated Columns)
* Data Modeling (Fact & Dimension Tables)

---

## 🔄 Data Workflow

### 1. Data Cleaning & Transformation (Power Query)

* Removed nulls & inconsistent values
* Standardized date formats
* Created structured columns (Month, Year, etc.)
* Ensured data quality for analysis

---

### 2. Data Modeling

* Built relationships between multiple tables:

  * Orders (Fact Table)
  * Customers
  * Restaurants
  * Date Table
* Applied **Star Schema approach**
* Created a **custom Date Table** for time intelligence

---

### 3. DAX Implementation

Created key measures:

* Total Revenue
* Total Orders
* Average Order Value
* Average Delivery Time

Additional logic:

* Top N filtering (Top 10 Restaurants)
* Conditional formatting for status insights
* Ranking logic using DAX

---

## 📈 Dashboard Features

### 🔹 KPI Cards

* Total Revenue
* Total Orders
* Avg Order Value
* Avg Delivery Time

### 🔹 Visual Insights

* Revenue Trend (Time-based analysis)
* Orders Trend
* Revenue by City (Top contributors)
* Top 10 Restaurants by Revenue
* Order Status Breakdown (Delivered, Cancelled, Delayed)
